# vue-covid19

## Website that shows basic Covid19 info worldwide, by country and by day

## Made in Vue.js

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
